# Anki Arquivologia

Aplicativo Android simples e offline com cartões cirúrgicos de Arquivologia.

## Cartões incluídos

1. Arquivo permanente: guarda definitiva, consulta e pesquisa.
2. Quatro critérios clássicos de classificação dos arquivos.
3. Conceito de arquivo e vínculo orgânico.

## Compilação

O APK de depuração é gerado automaticamente pelo GitHub Actions a cada alteração na branch `main`.
